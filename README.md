# JSP 웹 프론트엔드 포트폴리오

JSP 기반 웹 애플리케이션 실습 결과물 중 **프론트엔드(화면) · 구현 코드 · DB 연동 결과**를 정리한 포트폴리오입니다.
Apache Tomcat 서버에 직접 배포하여 동작시킨 실제 웹 화면 스크린샷을 담고 있습니다.

> 기업 입사 지원 제출용으로 정리되었습니다. (작업 기간: 2026-05 ~ 2026-06)

---

## 🛠 사용 기술

| 구분 | 기술 |
|------|------|
| 언어 | Java, JSP (Scriptlet / Expression / Declaration / Directive) |
| 마크업/스타일 | HTML5, CSS3, JavaScript |
| 서버 | Apache Tomcat 11 |
| 데이터베이스 | MySQL (JDBC 연동) |
| 도구 | VS Code, Git |

## 📌 프로젝트별 소개

### 1. 기상청 동네예보 대시보드 (`01_날씨예보_기상청동네예보`)
- 기상청 동네예보 데이터를 격자 좌표(X=61, Y=123) 기반으로 조회·시각화
- **다크 테마 대시보드**: 현재 기온(22°), 습도(80%), 풍속(2.7m/s) 카드 + 시간대별 차트 카드 구성
- `weather_dashboard.jsp`, `weather_forecast.jsp`, `weather_data_json.jsp` 등으로 구성, 10분 자동 갱신
- CSS 스타일링(`Malgun Gothic`, 카드 레이아웃, 표 디자인)을 직접 작성
- 🌟 대표 화면: `00_대표화면_Highlights/02_기상청동네예보_대시보드.png`

### 2. JSP 기초 · 표현식 · 회원/로그인 (`02_JSP기초_표현식_회원로그인`)
- JSP 핵심 문법 실습: Scriptlet `<% %>`, Expression `<%= %>`, Declaration `<%! %>`, page directive
- 함수/변수 선언, String·Integer 연산, 조건 처리, 회원/비회원 분기 및 폼(form) 데이터 전송(`input.jsp`)
- `es1.jsp ~ es9.jsp`, `member.jsp`, `login.html` 실습
- 🌟 대표 화면: `00_대표화면_Highlights/03_JSP표현식_연산결과_es5.png`

### 3. 성적관리 DB CRUD (`03_성적관리_DB_CRUD`)
- `examtable`(이름 / 학번(PK) / 국어 / 영어 / 수학)을 대상으로 한 **CRUD 전체 구현**
  - 테이블 생성 · 삭제 · 데이터 입력 · 전체 조회 · 단건 조회 · 정정/삭제
- MySQL **JDBC 연동**, 프레임 분할 레이아웃(좌측 메뉴 / 우측 내용)
- 동일 기능을 두 가지 디자인으로 제작: 기본형(`DB01`) + 고급 스타일형(`DBAI`)
- 🌟 대표 화면: `05_성적관리_전체조회_테이블.png`, `06_성적관리_성적입력폼.png`, `04_성적관리_DBAI_소개화면.png`, `07_성적관리_DB01_메뉴화면.png`

### 4. 무료와이파이 위치 조회 (`04_무료와이파이_FreeWiFi`)
- 공공 무료와이파이 위치 데이터(위도/경도 포함)를 조회하는 웹 화면
- **페이징 처리**(page / cnt 파라미터) 직접 구현, MySQL 연동(`jdbc:mysql://localhost:3306/kopo14`)
- `FW1.jsp ~ FW3.jsp`, 다크 테마 위치 검색 UI
- 🌟 대표 화면: `00_대표화면_Highlights/08_무료와이파이_페이징조회.png`
